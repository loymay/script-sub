# MMSUB 脚本合集管理器

个人自用的 Linux 脚本快捷调取合集，集合了常用的多功能脚本、代理工具、自用管理脚本等。

## 🚀 快速开始

在您的 Linux 终端中运行以下一键脚本：

```bash
wget -O mmsub.sh https://raw.githubusercontent.com/loymay/script-sub/main/mmsub.sh && chmod +x mmsub.sh && ./mmsub.sh
```

## 🛠️ 功能特点

- **精美 UI**: 采用双列并排设计，色彩分明。
- **快捷指令**: 安装后可直接通过 `mmsub` 命令唤起。
- **分类清晰**:
  - 【多功能聚合】: 科技lion、老王SSH工具箱等。
  - 【代理脚本】: Sing-box (lite/fscarmen/eooce)、ArgoX、MTP 等。
  - 【常用工具】: IP质量查询、Warp (fscarmen/Misaka) 等。
  - 【自用脚本】: 个人 SSB Singbox 管理。
- **安全卸载**: 运行脚本中的 `88` 选项即可卸载本管理工具，不影响通过本工具安装的其他脚本。
